# Taskify 📋✨

## Project Description 🚀

Taskify is a web application for task management, built using React.js. The app allows users to add, edit, delete, and filter tasks by various statuses.

## Features 🌟

- Adding new tasks ➕
- Updating task status 🔄
- Deleting tasks 🗑️
- Filtering tasks by status (New, In Progress, Completed) 🔍

## Technologies 💻

- React.js
- Vanilla JavaScript for API requests
- CSS Modules
- Backend: Node.js & Express.js
- Database: MongoDB

## Backend Architecture 🌐

- API service developed with Node.js and Express.js
- Database interactions managed through MongoDB
- RESTful API endpoints for task management

## Installation 🔧

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the app: `npm start`

## API Interaction 🔗

Communicates with backend server at:
`https://taskify-api-mq44.onrender.com`
Handles CRUD operations for tasks via RESTful API

## Project Structure 📂

- `App.jsx`: Main component
- `components/`: React components
- `api/`: API request functions
